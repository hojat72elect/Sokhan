### Tech Stack

- [Ionic](https://ionic.io/) - A cross-platform UI toolkit focusing on mobile and desktop app development via standard
  web technologies such as HTML, CSS, and JavaScript.
- [React](https://react.dev/) - A component-based JavaScript library that provides a virtual DOM and has an extensive
  open source ecosystem of tools.
- [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) - Most commonly used programming language of the
  web.
- [TypeScript](https://www.typescriptlang.org/) - A safety-net for JavaScript which adds static types and makes catching
  erros and debugging so much easier.
- [Vite JS](https://vite.dev/) - A frontend build tool for near-instant server start and Hot Module Reloading.
